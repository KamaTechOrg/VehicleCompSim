Detailed Description

template<AesVariant Aes_var>
class aes::AES_block_encrypt_impl< Aes_var >

AES block encryption implementation for a specific AES variant (128-bit, 192-bit, 256-bit).

This class implements AES encryption and decryption for the specified AES variant and provides methods for key expansion, round operations, and other internal AES transformations.

Template Parameters
Aes_varThe AES variant to be used (Aes128, Aes192, or Aes256).
