Detailed Description

Provides AES block encryption modes such as ECB, CBC, and CTR.

This class offers static methods for encrypting and decrypting messages using different AES block encryption modes, including ECB (Electronic Codebook), CBC (Cipher Block Chaining), and CTR (Counter Mode). It supports both standard and SYCL-enabled modes.
